# Arduino library for I2C LCD
Support both 2x16 LCD and 4x20 LCD

Wiring:</br>
Ardino ----------------------- I2C LCD</br>
====================================</br>
GND ----------------------------> GND</br>
Vcc ----------------------------> VCC</br>
A4 or SDA -----------------------> SDA</br>
A5 or SCL -----------------------> SCL</br>
